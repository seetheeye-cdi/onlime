요즘 ClaudeCode를 이리저리 사용해보다가 hook 기능이 있다는 것을 알게 되었습니다. 아는 개발자 친구한테 물어보니, 자기는 OpenAI Codex의 작업이 끝나면 OpenAI Codex가 다시 켜지는 것을 이용하고 있다고 해서, 저도 한 번 구현해보았습니다.

ClaudeCode 작업 끝나면 자동으로 알려주기
1. Apple Shortcuts 설정하기

Apple Shortcuts에 위와 같이 Shortcuts를 만듭니다. 이름은 Claude Finished로 설정했습니다.

Mac 알림 센터에 알림을 허용 후

Pause on Mac으로 하고 있언 일을 잠시 멈춘 후

터미널(Warp를 이용해서 Warp로 설정)을 열어 ClaudeCode를 보게 설정

위에 ▶︎를 클릭하면 터미널이 켜지는 것을 확인해 볼 수 있습니다.

2. notify.sh와 hook 설정
~/.claude/notify.sh 경로에 아래와 같이 설정, Shortcuts의 Claude Finished를 실행하도록 설정했습니다.


사본
#!/bin/zsh
/usr/bin/shortcuts run "Claude Finished"
그리고 hook을 이용해 ClaudeCode 세션이 끝날 때를 hook으로 잡아, 세션이 끝나면 notify.sh를 실행되도록 설정했습니다.

즉, ClaudeCode 세션이 끝나면 자동으로 Shortcuts 워크플로우가 켜져서, ClaudeCode를 바로 키게 하는 방식입니다.

이러면 설정은 끝입니다.

3. 결과물
이제, 유튜브를 보는 등 다른 활동을 하다가 ClaudeCode 세션이 끝나면 자동으로 진행하고 있는 것을 멈추고 ClaudeCode를 다시 실행할 수 있게 되었습니다.
